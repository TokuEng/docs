## Overview

Retrieves complete details for an organization, including name, configuration, and associated settings.

## Endpoint

```
GET /api/payroll/getOrgDetails
```

## Authentication

Requires `tokuOrgAdmin` role for the specified organization.

## Request

### Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `orgID` | number | Yes | The unique identifier of the organization |

### Example Request

```bash
curl -X GET "https://api.toku.com/api/payroll/getOrgDetails" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"orgID": 100}'
```

### JavaScript Example

```javascript
const response = await fetch('https://api.toku.com/api/payroll/getOrgDetails', {
  method: 'GET',
  headers: {
    'Authorization': `Bearer ${process.env.TOKU_API_KEY}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    orgID: 100
  })
});

const organization = await response.json();
```

## Response

### Success Response (200)

```json
{
  "result": {
    "data": {
      "orgID": 100,
      "name": "Acme Corporation",
      "logoFileID": 250,
      "settings": {
        "defaultCurrencyID": 1,
        "payrollEnabled": true,
        "contractorPaymentsEnabled": true
      },
      "createdAt": "2023-06-01T00:00:00.000Z",
      "updatedAt": "2024-01-10T08:30:00.000Z"
    }
  }
}
```

### Response Fields

| Field | Type | Description |
|-------|------|-------------|
| `orgID` | number | Unique organization identifier |
| `name` | string | Organization display name |
| `logoFileID` | number | File ID of organization logo (if set) |
| `settings` | object | Organization configuration settings |
| `createdAt` | string | Record creation timestamp |
| `updatedAt` | string | Last update timestamp |

### Error Responses

| Status | Code | Description |
|--------|------|-------------|
| 401 | `UNAUTHORIZED` | Invalid or missing API key |
| 403 | `FORBIDDEN` | User lacks permission for this organization |
| 404 | `NOT_FOUND` | Organization ID does not exist |
