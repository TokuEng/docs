## Overview

Creates a new employee record in the payroll system. The employee will need additional configuration (pay profile, tax profile) before they can be included in payroll runs.

## Endpoint

```
POST /api/payroll/createEmployee
```

## Authentication

Requires `tokuOrgAdmin` role for the organization.

## Request

### Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `orgID` | number | Yes | Organization ID to add employee to |
| `firstName` | string | Yes | Employee's first name |
| `lastName` | string | Yes | Employee's last name |
| `addressLine1` | string | No | Primary address (default: "No Address Given") |
| `addressLine2` | string | No | Secondary address line |
| `addressLine3` | string | No | City, state, postal code |
| `startDate` | date | No | Employment start date |
| `endDate` | date | No | Employment end date (for fixed-term contracts) |

### Example Request

```bash
curl -X POST "https://api.toku.com/api/payroll/createEmployee" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "orgID": 100,
    "firstName": "Jane",
    "lastName": "Smith",
    "addressLine1": "456 Oak Avenue",
    "addressLine2": "Suite 200",
    "addressLine3": "San Francisco, CA 94102",
    "startDate": "2024-02-01T00:00:00.000Z"
  }'
```

### JavaScript Example

```javascript
const response = await fetch('https://api.toku.com/api/payroll/createEmployee', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${process.env.TOKU_API_KEY}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    orgID: 100,
    firstName: 'Jane',
    lastName: 'Smith',
    addressLine1: '456 Oak Avenue',
    addressLine2: 'Suite 200',
    addressLine3: 'San Francisco, CA 94102',
    startDate: '2024-02-01T00:00:00.000Z'
  })
});

const employee = await response.json();
```

## Response

### Success Response (200)

```json
{
  "result": {
    "data": {
      "employeeID": 501,
      "orgID": 100,
      "firstName": "Jane",
      "lastName": "Smith",
      "addressLine1": "456 Oak Avenue",
      "addressLine2": "Suite 200",
      "addressLine3": "San Francisco, CA 94102",
      "startDate": "2024-02-01T00:00:00.000Z",
      "endDate": null,
      "status": "ACTIVE",
      "createdAt": "2024-01-20T10:00:00.000Z"
    }
  }
}
```

### Error Responses

| Status | Code | Description |
|--------|------|-------------|
| 400 | `INVALID_INPUT` | Invalid or missing required fields |
| 401 | `UNAUTHORIZED` | Invalid or missing API key |
| 403 | `FORBIDDEN` | User lacks permission for this organization |
| 409 | `ALREADY_EXISTS` | Employee with same details already exists |

## Next Steps

After creating an employee, you'll need to:

<Steps>
  <Step title="Configure Pay Profile">
    Set up salary, pay frequency, and compensation details
  </Step>
  <Step title="Set Tax Profile">
    Configure tax withholding based on employee's jurisdiction
  </Step>
  <Step title="Verify Eligibility">
    Use `canRunPayrollForEmployee` to verify the employee can receive payroll
  </Step>
  <Step title="Include in Payroll">
    Add the employee to your next payroll run
  </Step>
</Steps>

<Note>
Employees cannot be included in payroll until their pay profile and tax profile are configured. Contact Toku support for assistance with employee setup.
</Note>
