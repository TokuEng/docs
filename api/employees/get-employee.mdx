## Overview

Retrieves complete details for an employee, including personal information, employment status, and payroll configuration.

## Endpoint

```
GET /api/payroll/getEmployeeDetails
```

## Authentication

Requires `employeeAdmin` role for the specific employee, or `orgAdmin`/`tokuOrgAdmin` for the organization.

## Request

### Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `employeeID` | number | Yes | The unique identifier of the employee |

### Example Request

```bash
curl -X GET "https://api.toku.com/api/payroll/getEmployeeDetails" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"employeeID": 500}'
```

### JavaScript Example

```javascript
const response = await fetch('https://api.toku.com/api/payroll/getEmployeeDetails', {
  method: 'GET',
  headers: {
    'Authorization': `Bearer ${process.env.TOKU_API_KEY}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    employeeID: 500
  })
});

const employee = await response.json();
```

```json
{
  "result": {
    "data": {
      "employeeID": 500,
      "orgID": 100,
      "firstName": "John",
      "lastName": "Doe",
      "addressLine1": "123 Main Street",
      "addressLine2": "Apt 4B",
      "addressLine3": "New York, NY 10001",
      "startDate": "2023-01-15T00:00:00.000Z",
      "endDate": null,
      "status": "ACTIVE",
      "payProfile": {
        "payrollPeriodsPerYear": 12,
        "salaryComponents": [
          {
            "yearlySalary": 120000,
            "currencyID": 1,
            "salaryFrequency": "YEARLY"
          }
        ]
      },
      "taxProfile": {
        "calculatorVersionID": 5,
        "jurisdiction": "US-NY"
      },
      "createdAt": "2023-01-10T09:00:00.000Z",
      "updatedAt": "2024-01-15T14:30:00.000Z"
    }
  }
}
```

### Response Fields

| Field | Type | Description |
|-------|------|-------------|
| `employeeID` | number | Unique employee identifier |
| `orgID` | number | Organization ID |
| `firstName` | string | Employee's first name |
| `lastName` | string | Employee's last name |
| `addressLine1` | string | Primary address line |
| `addressLine2` | string | Secondary address line |
| `addressLine3` | string | City, state, postal code |
| `startDate` | string | Employment start date |
| `endDate` | string | Employment end date (null if active) |
| `status` | string | Employment status: `ACTIVE`, `INACTIVE`, `TERMINATED` |
| `payProfile` | object | Salary and pay period configuration |
| `taxProfile` | object | Tax calculation settings |
| `createdAt` | string | Record creation timestamp |
| `updatedAt` | string | Last update timestamp |

### Error Responses

| Status | Code | Description |
|--------|------|-------------|
| 401 | `UNAUTHORIZED` | Invalid or missing API key |
| 403 | `FORBIDDEN` | User lacks permission for this employee |
| 404 | `NOT_FOUND` | Employee ID does not exist |
