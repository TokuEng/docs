## Introduction

The Toku API enables third-party developers to integrate payroll processing, employee management, and contractor payments into their applications. Our RESTful API provides programmatic access to core Toku functionality.

## Base URL

All API requests should be made to:

```
https://api.toku.com/api
```

## API Versioning

The current API version is **v1**. The version is included in the endpoint path where applicable.

## Key Features

- **Multi-currency Support**: Process payroll in fiat and cryptocurrencies including USDC, USDT, and DAI
- **Token Compensation**: Support for token vesting schedules and bonus payments
- **Tax Calculations**: Automated tax withholding based on employee jurisdiction
- **Real-time Exchange Rates**: Integration with market data for accurate crypto conversions

## Response Format

All responses are returned in JSON format. Successful responses include the requested data, while errors include descriptive messages.

```json
{
  "result": {
    "data": {
      // Response payload
    }
  }
}
```
