## Overview

Returns a list of all currencies supported by Toku for payroll and payments, including both fiat currencies and cryptocurrencies.

## Endpoint

```
GET /api/payroll/getAllCurrencies
```

## Authentication

Requires `tokuOrgAdmin` role.

## Request

### Parameters

This endpoint does not require any parameters.

### Example Request

```bash
curl -X GET "https://api.toku.com/api/payroll/getAllCurrencies" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

### JavaScript Example

```javascript
const response = await fetch('https://api.toku.com/api/payroll/getAllCurrencies', {
  method: 'GET',
  headers: {
    'Authorization': `Bearer ${process.env.TOKU_API_KEY}`,
    'Content-Type': 'application/json'
  }
});

const currencies = await response.json();
```

### Response Fields

| Field | Type | Description |
|-------|------|-------------|
| `currencyID` | number | Unique currency identifier (use this in other API calls) |
| `name` | string | Full currency name |
| `currencyCode` | string | ISO currency code or ticker symbol |
| `crypto` | boolean | Whether this is a cryptocurrency |
| `isActive` | boolean | Whether the currency is currently active |
| `source` | string | Data source for the currency |
| `sourceCurrencyID` | number | ID in the source system (for crypto) |
| `coinLogoUrl` | string | URL to currency logo (for crypto) |

### Error Responses

| Status | Code | Description |
|--------|------|-------------|
| 401 | `UNAUTHORIZED` | Invalid or missing API key |
| 403 | `FORBIDDEN` | User lacks required permissions |

## Supported Currency Types

### Fiat Currencies

Common fiat currencies include:
- USD (US Dollar)
- EUR (Euro)
- GBP (British Pound)
- And many more...

### Cryptocurrencies

Supported stablecoins and cryptocurrencies:
- USDC (USD Coin)
- USDT (Tether)
- DAI (Dai)
- BTC (Bitcoin)
- ETH (Ethereum)
- And more...

<Tip>
Cache this response as currency lists change infrequently. Refresh daily or when you encounter an unknown currency ID.
</Tip>

## Use Cases

### Building a Currency Selector

```javascript
async function getCurrencyOptions() {
  const response = await fetch('https://api.toku.com/api/payroll/getAllCurrencies', {
    headers: {
      'Authorization': `Bearer ${process.env.TOKU_API_KEY}`,
      'Content-Type': 'application/json'
    }
  });

  const { result: { data: currencies } } = await response.json();

  // Separate fiat and crypto
  const fiatCurrencies = currencies.filter(c => !c.crypto && c.isActive);
  const cryptoCurrencies = currencies.filter(c => c.crypto && c.isActive);

  return { fiatCurrencies, cryptoCurrencies };
}
```
