## What is TGA?

Toku TGA (Token Grant Administration) is a platform for managing token-based compensation programs. It enables organizations to:

- **Create token grants** for employees, contractors, and advisors
- **Configure vesting schedules** including cliff, linear, and milestone-based vesting
- **Process distributions** of vested tokens to recipients
- **Track and report** on grant activity for compliance and accounting

## Key Features

### For Administrators

| Feature | Description |
|---------|-------------|
| **Grant Management** | Create, modify, and track token grants |
| **Vesting Schedules** | Flexible vesting configurations |
| **Bulk Operations** | Import multiple grants via CSV |
| **Distribution Processing** | Single or batch token distributions |
| **Reporting** | Export data for compliance and accounting |

### For Recipients

| Feature | Description |
|---------|-------------|
| **Grant Dashboard** | View all grants in one place |
| **Vesting Tracker** | See vesting progress and upcoming dates |
| **Token Claims** | Claim vested tokens to your wallet |
| **History** | View past claims and transactions |
