## Overview

This guide walks you through creating your first project with the library. By the end, you'll have a working example you can build upon.

## Step 1: Create a New Project

Create a new directory and initialize a project:

```bash
mkdir my-project
cd my-project
npm init -y
```

## Step 2: Install the Library

```bash
npm install your-library
```

## Step 3: Create Your First File

Create an `index.js` file:

```javascript
import { createClient } from 'your-library';

// Initialize the client
const client = createClient({
  // Add your configuration here
});

// Use the library
async function main() {
  const result = await client.doSomething();
  console.log(result);
}

main();
```

## Step 4: Run Your Project

```bash
node index.js
```

You should see output in your terminal confirming everything works.
