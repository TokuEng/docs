## Key features

### Token Grants
Create grants with customizable vesting schedules. Support for cliff vesting, linear vesting, and milestone-based vesting.

### Token Payroll
Pay employees and contractors in tokens or stablecoins. Integrates with your existing HRIS for seamless payroll processing.

### Distributions
Process and track token distributions to recipients. Batch processing for efficient multi-recipient distributions.

### Compliance
Built-in KYC verification and tax withholding to keep your program compliant.
