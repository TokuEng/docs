# Build your token compensation program with TGA

TGA (Token Grant Administration) is Toku's comprehensive platform for managing token-based compensation. From creating grants and configuring vesting schedules to processing payroll and settling tokens - TGA handles the entire lifecycle of token compensation.

## Core Features

<CardGroup cols={2}>
  <Card title="Token Grants" icon="gift" href="/features/grants/overview">
    Create and manage token grants with flexible vesting schedules. Support for RTUs, Options, Warrants, and 10+ grant types.
  </Card>
  <Card title="Token Payroll" icon="money-bill-transfer" href="/features/payroll/overview">
    Pay employees in tokens or stablecoins with HRIS integration, exchange rate management, and automated settlements.
  </Card>
  <Card title="Wallet Management" icon="wallet" href="/features/wallets/overview">
    Embedded wallets for easy onboarding, custody provider integration, and multi-sig support.
  </Card>
  <Card title="Compliance" icon="shield-check" href="/features/compliance/kyc">
    Built-in KYC verification, tax withholding calculations, and regulatory compliance tools.
  </Card>
</CardGroup>
