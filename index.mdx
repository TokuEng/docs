## Overview

Toku is the API for stablecoin payroll, enabling companies to pay employees and contractors globally using fiat currency or stablecoins. Our platform handles compliance, tax withholding, and benefits administration across 100+ countries.

## Quick Example

```javascript
import { Toku } from '@toku/sdk';

const toku = new Toku({
  apiKey: process.env.TOKU_API_KEY,
});

// Run payroll for your team
const payroll = await toku.payroll.run({
  employeeId: 'emp_123',
  amount: 5000,
  currency: 'USDC',
});

console.log(`Payroll processed: ${payroll.id}`);
```
