# Toku: ADP Export Employees

This is a guide to export Position ID and Associate IDs from ADP Workforce Now

1. Go to [https://workforcenow.cloud.adp.com/theme/index.html#/home](https://workforcenow.cloud.adp.com/theme/index.html#/home)
2. Select “Reports & Analytics”
    
    ![Screenshot 2025-12-04 at 1.54.44 PM.png](Screenshot_2025-12-04_at_1.54.44_PM.png)
    
3. Select “All Standard Reports”
    
    ![Screenshot 2025-12-04 at 1.57.03 PM.png](Screenshot_2025-12-04_at_1.57.03_PM.png)
    
4. Select “Personal & Employment” then “Employee Information”
    
    ![Screenshot 2025-12-04 at 1.58.34 PM.png](Screenshot_2025-12-04_at_1.58.34_PM.png)
    
5. Select “What’s Displayed on this Report”
    
    ![Screenshot 2025-12-04 at 2.00.39 PM.png](Screenshot_2025-12-04_at_2.00.39_PM.png)
    
6. Select the following fields
    1. Pay Group
    2. Position ID
    3. Associate ID
    4. Name
    5. Work E-Email
    6. Hit “Save”
        
        ![Screenshot 2025-12-04 at 2.03.09 PM.png](Screenshot_2025-12-04_at_2.03.09_PM.png)
        
7. Select “Appearance and Other Settings”
    
    ![Screenshot 2025-12-04 at 2.04.39 PM.png](Screenshot_2025-12-04_at_2.04.39_PM.png)
    
8. Select the following fields
    1. Pay Group
    2. Position ID
    3. Associate ID
    4. Name
    5. Work Email (If available)
    6. Hit “Save”
        
        ![Screenshot 2025-12-04 at 2.06.49 PM.png](Screenshot_2025-12-04_at_2.06.49_PM.png)
        
9. Select “Run as Excel”
    
    ![Screenshot 2025-12-04 at 2.11.01 PM.png](Screenshot_2025-12-04_at_2.11.01_PM.png)
    
10. Select “View as XLSX” to download
    
    ![Screenshot 2025-12-04 at 2.10.05 PM.png](Screenshot_2025-12-04_at_2.10.05_PM.png)
